무료 API로 찾아보기(openAPI)

TMDB, 알라딘

사용자가 정상 접근인지 판별 어려움(한 명이 동시에 100만개 계정)
=> API KEY를 활용하여 인증 확인, 사용자 확인

한 명의 사용자가 계속적으로 100만번 => 사용량 제한

json: API가 사용하는 데이터 형식, 데이터를 저장 전송할 때 많이 사용되는 경량의 텍스트 기반의 데이터 형식(통일되지 않은 데이터 구조가 많아서)
  - 파싱: 데이터를 의미있는 구조로 분석하고 해석하는 과정
  - json.loads(): JSON 형식의 문자열을 파싱하여 python Dictionary로 변환

외부API를 불러 오면 서비스의 질을 능동적으로 조절하기 어려움

API키는 3.0 아니라 Current로

  - 과학적 방법론, 프로세스, 알고리즘, 시스템 동원

**판다스(원하는 데이터만 추출, 데이터를 분석): 판다스에 없는 유연석을 제공, 고성능의 데이터 구조를 만들 수 있음
