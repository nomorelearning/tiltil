- 패턴매칭 : 텍스트에서 패턴이 존재하는 모든 위치 찾는 문제
텍스트 -> t[], 패턴 -> p[]
텍스트의 길이 -> n, 패턴의 길이 -> m
텍스트의 특정 위치 -> i, 패턴의 특정 위치 -> j
패턴 매칭 알고리즘을 볼 때
일치 t[i] == p[j] => 다음 것 비교(+1 씩) (패턴의 길이m만큼 일치할 때까지 ; j == m)
불일치 t[i] != p[j] ** 불일치 했을 때 i와 j를 어떻게 설정하냐가 각 배치방법의 특징이 됨
